Upgrading Libraries

If you were using an earlier version of TapLynx...

TapLynx is now just one library

In the past, the TapLynx library was two libraries — one for simulator and one device. Now there’s just one libTapLynx library (which works for both simulator and device).

Quattro no longer supported

Though Quattro ads were supported in earlier versions of TapLynx, they are no longer supported — since Quattro was acquired by Apple, and they’re concentrating on iAds instead. (We’re on the case. It’s high on the to-do list.)

If you still need Quattro ads, you can use TapLynx 1.3.4 and Xcode 3.2.2 (or earlier).

Medialets and Flurry

There are new versions of the libraries for Medialets and Flurry which are compatible with iOS 4. You may need to download those and replace your current copies with the new versions.

See their respective websites for the new versions of the libraries.
