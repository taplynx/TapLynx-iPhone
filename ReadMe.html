Welcome to TapLynx

TapLynx is a library that makes building news and media iPhone apps very easy.

No programming is required. Configure the application via a property list file, add artwork and feeds, and build it.

This means you can iterate over the design, not the programming.

Getting Started

Open the iPhoneApp project in Xcode. Type cmd-R to build it and run it in the Simulator. It should just work, as long as you’re using iPhone SDK 3.2.5 or greater and the iOS 4.2 SDK.

See the TapLynx wiki for more documentation on getting started, configuring your app, adding artwork, setting up feeds, and so on. The documentation is evolving and ongoing, so if there’s something not yet answered there, please ask about it on the TapLynx Google Group.

Note about Xcode and SDKs

You should build using Xcode 4.2.1 or newer targeting iOS 4.3 or newer. It will create apps that run on iOS 4 and iOS 5.
