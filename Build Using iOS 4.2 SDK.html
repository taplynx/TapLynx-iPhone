Build using iOS 4.2 SDK

You should build using the iOS 4.2 SDK.

The iPhone OS Deployment Target should be iPhone OS 3.0 — so that your apps can still run on earlier iPhones and on iPads.

The Base SDK should be iPhone Device 4.2. You should use Xcode 3.2.5 or greater.
